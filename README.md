# eslint-config

## Requirements

- [Node.js](https://nodejs.org)
- [NPM](https://www.npmjs.com)

## Installation

```console
$ npm install --save-dev aminnairi/eslint-config
```

## Usage

```console
$ touch .eslintrc.js
```

```javascript
"use strict";

module.exports = {
    "extends": [
        "@aminnairi"
    ]
};
```
